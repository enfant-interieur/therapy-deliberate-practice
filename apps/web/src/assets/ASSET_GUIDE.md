# Patient Rig Asset Guide

Replace placeholder rig layers with transparent PNGs.

## Required layers

- `head.png`
- `eyes.png`
- `mouth.png`
- `body.png`
- `hands.png`

## Guidelines

- Use the same canvas size (520x360) with consistent anchor points.
- Keep transparent margins to preserve tweening.
- Export at 2x for retina displays.
